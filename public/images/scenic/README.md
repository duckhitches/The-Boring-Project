# Scenic Background Images

This folder contains the scenic background images for the application's theme system.

## Required Images

To enable the scenic theme, add these images to this folder:

1. **mountain-lake.jpg** - A beautiful mountain lake scene
2. **forest-path.jpg** - A peaceful forest path
3. **ocean-sunset.jpg** - A stunning ocean sunset
4. **city-skyline.jpg** - A modern city skyline

## Image Requirements

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 or higher
- **File Size**: Keep under 2MB for optimal loading
- **Quality**: High quality, visually appealing images

## Usage

Once you add your images with the exact names listed above, users will be able to:
- Select the scenic theme in Settings
- Choose from the 4 pre-uploaded images
- Have their selection saved and restored

## File Structure

```
public/
  images/
    scenic/
      mountain-lake.jpg
      forest-path.jpg
      ocean-sunset.jpg
      city-skyline.jpg
```

The application will automatically load these images when users select the scenic theme.
